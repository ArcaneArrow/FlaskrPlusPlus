# Flaskr Plus Plus
Authors: Emory Cooley, Mark Liffiton, Armin Ronacher, and the Flask Team<br>
Date: 11/08/2025<br>
Name: CS253 Flaskr Plus Plus<br>

A Flask application adapted from Flaskr in the [Flask 0.12
tutorial](https://www.iwu.edu/~mliffito/flask_tutorial/).

Flaskr is a demonstration app from the Flask codebase.
Flaskr Plus is a modification of it that adds a few simple
features such as post categories and deletion.
Flaskr Plus Plus is a modification of it that lets users edit their posts.
